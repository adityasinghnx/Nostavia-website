SDK & API Documentation

Welcome to the Nostavia developer platform. Our SDK and API allow you to integrate the Whole-Body Intelligence System — lab decoding, food bio-impact analysis, wearable connectivity, adaptive protocols, and the AI health concierge — into your product through a single integration.

Intelligence Sync

The /v1/intelligence/sync endpoint is the core of Nostavia. It pulls data from all connected sources for a given user, runs it through the SOMA intelligence layer, and returns a unified health intelligence payload.

POST https://api.nostaviahealth.com/v1/intelligence/sync

Request Body

Pass the user ID, the intelligence modules you want to activate, the data sources, and the date range.

{
  "user_id": "usr_9a8b7c6d",
  "data_sources": ["oura", "cgm", "labcorp"],
  "date_range": {
    "start": "2023-10-01",
    "end": "2023-10-07"
  },
  "metrics": [
    "glucose_variability",
    "resting_hr",
    "hba1c"
  ]
}

Response

The response includes the processing status, the calculated health intelligence score, and AI-generated clinical insights.

{
  "status": "normalized",
  "health_intelligence_score": 84,
  "insights": [
    "Glucose variability improved by 12%.",
    "Biomarker drift detected: Vitamin D trending below optimal range."
  ]
}
